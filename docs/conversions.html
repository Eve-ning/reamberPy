Conversions

Converting is rather difficult, especially between maps that use millisecond and fractional snapping. If there are any issues, please do report them.

Syntax

The syntax is always consistent.

There are many combinations, I will only show a few.

from reamber.quaver.QuaMap import QuaMap from reamber.algorithms.convert import QuaToSM qua = QuaMap.read_file("file.qua") sm = QuaToSM.convert(qua) sm.write_file("file.sm")
from reamber.osu.OsuMap import OsuMap from reamber.algorithms.convert import OsuToQua osu = OsuMap.read_file("file.osu") qua = OsuToQua.convert(osu) qua.write_file("file.qua")
from reamber.o2jam.O2JMapSet import O2JMapSet from reamber.algorithms.convert import O2JToOsu o2j = O2JMapSet.read_file("file.ojn") osu = O2JToOsu.convert(o2j) osu.write_file("file.osu")
from reamber.bms import BMSMap from reamber.algorithms.convert import BMSToOsu bms = BMSMap.read_file("file.bme") osu = BMSToOsu.convert(bms) osu.write_file("file.osu")

Note that converting to a game that uses MapSet, it will automatically cast any Map to a MapSet.

Most conversions will yield a bad offset, please do check them.
